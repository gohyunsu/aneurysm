Physics-Based Simulations with Masked Graph Neural Networks (ICLR 2025)를 large-mesh CFD pretraining 관점에서 읽는 note.

HubDatasetsMethodsLiteratureOriginal paper

ONE-SENTENCE IDEA

Hide part of a mesh so the representation must encode global structure.

MeshMask는 input mesh node 일부를 mask하고, asymmetric encoder–decoder가 masked node를 포함한 full-state prediction을 수행하도록 학습한다. 목적은 local neighbor shortcut을 줄이고, large/heterogeneous CFD data에서 pretraining과 long rollout robustness를 높이는 것이다.

ADOPTION PLAN

  1. Do not start with masking

    먼저 BenchAnXplore에서 unmasked MGN/In-PI baseline과 memory profile을 재현한다.

  2. Define mask semantics

    geometry node를 제거하는지, state feature를 mask하는지, boundary node는 mask 가능한지에 따라 task가 달라진다. wall/inlet/outlet rule을 manifest에 남긴다.

  3. Test transfer honestly

    synthetic pretraining과 patient-derived testing을 섞지 않는다. pretraining corpus, fine-tune budget, OOD geometry split을 분리 report한다.
