01 / BENCHANXPLORE

Transient CFD benchmark for surrogate learning

Verified acquisition

Archive: 1,123,362,206 bytes (약 1.12 GB)
Integrity: unzip -t → No errors detected
Extracted: HDF5 105개, XDMF 105개
Representative case: MESH_193-1

이 archive는 In-PI-MGN 재현용 CFD field source이다. 파열 label·임상 metadata dataset이 아니므로 risk model의 환자 cohort로 취급하지 않는다.

해석: data0은 3차원 좌표 후보이고 data1은 4-node cell connectivity 후보이다. 그 뒤에는 3-vector와 1-scalar 배열이 81회 반복된다. 단, HDF5 key에 velocity/WSS 등의 semantic name이 없으므로 dataN만으로 물리량 이름을 단정하지 않는다. XDMF Attribute와 배포 논문의 reader를 파싱한 뒤 field_registry.csv로 확정한다.

03 / ACQUISITION QUEUE

Downloaded, queued, and access-controlled sources

CMHA / Gong 2024

patients 9.99 GB + controls 4.49 GB + statistics CSV. 다운로드 대기 중이다. 완료 후 NIfTI/STL/JSON/CFD summary의 case key, units, missingness부터 inventory한다.

AneuriskData

Git mirror 약 1.36 GB. 다운로드 대기 중이다. surface/centerline/volume/morphology의 실제 case coverage를 비교한다. CFD가 있다고 가정하지 않는다.

AneuG-Flow

geometry archive 3.47 GB를 수집했다. full transient field corpus는 약 2.62 TB이므로 shared NFS에 무단 수집하지 않는다. storage owner 승인 뒤 selected/full tier를 결정한다.

IntrA · Aneumo

공개 repository를 수집했다. segmentation/shape data의 license·split·file count를 inventory한 뒤 clinical-risk dataset과 분리해 geometry benchmark로 기록한다.

Access-controlled sources ADAM, CADA, TopAneu는 별도 등록·약관 동의가 필요하다. 인증 없이 자동 수집하지 않으며, 사용자가 권한을 제공한 뒤에만 acquisition log를 추가한다.
