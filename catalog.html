INTEGRATION CONTRACT

Four layers, explicit joins

raw/archive + URL + license + SHA-256
extracted/publisher folder layout untouched
manifests/case/asset/field registry and missingness
derivatives/NIfTI/VTP/graph/features with version
experiments/split, config, metrics, figures
Never doInstead
copy every file under a guessed patient folderkeep raw paths; join only documented within-dataset keys
convert STL/VTP then delete source meshpreserve source; store conversion software/version/checksum
treat missing status as unrupturedretain missingness and restrict supervised cohort explicitly
mix real CFD and synthetic field as the same labelrecord simulation provenance and evaluate domain transfer separately
