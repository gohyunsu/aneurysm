01 / RESEARCH QUESTION

From image to flow,
without losing provenance.

이 연구는 CTA/MRA에서 동맥류를 찾고 해부학적 위치를 해석하는 문제와, CFD를 대체할 만큼 빠르게 혈류장을 예측하는 문제를 연결한다. 영상 label·실제 CFD·surrogate 예측값은 서로 다른 근거로 관리한다.

Working hypothesis임상·형태학 변수에 혈류역학 표현을 더하면 더 풍부한 위험 표현을 만들 수 있다. 이 가설은 환자·기관 분리 검증과 calibration을 통과해야만 의미가 있다.

02 / TASK DEFINITION

Three tasks, different evidence.

A

Clinical Imaging AI

CTA/MRA에서 동맥류 후보를 검출하고 병변 mask와 혈관 위치 class를 예측한다.

입력
3D CTA 또는 MRA
출력
lesion mask, vessel/location label
평가
lesion sensitivity, Dice, NSD, FP/case
B

Hemodynamics Surrogate

비정형 CFD mesh와 inlet context로부터 시간에 따른 velocity field를 예측하고 WSS·OSI를 계산한다.

입력
mesh graph, state, boundary context
출력
velocity rollout, WSS/OSI summary
평가
1/50-step RMSE, WSS error, mass flux
C

Risk Stratification

clinical, morphology, real/surrogate hemodynamics를 결합해 rupture status와의 연관성을 검증한다.

입력
환자별 multimodal feature
출력
calibrated research score
평가
AUROC/AUPRC, calibration, external split
Scope guardrail 파열/비파열 상태는 미래 파열 위험과 다르다. 이 사이트에서는 두 label을 동일한 임상 주장으로 바꾸지 않는다.

04 / INTEGRATION CONTRACT

Do not build a false “unified patient folder.”

환자 동일성·좌표계·단위가 검증되지 않은 파일을 한 폴더로 섞으면 provenance를 잃는다. 원본은 immutable하게 두고, manifest가 raw asset과 derivative를 연결한다.

01Raw archivechecksum, source URL, license, acquisition date
02Extracted source layout배포처의 폴더·파일명을 그대로 보존
03Asset manifestsource_case_id, file type, bytes, coordinate/unit, provenance
04Task derivativeNIfTI, VTP, graph, case-level feature를 versioned output으로 생성
05Split manifestsurrogate=geometry-disjoint / risk=patient·site-disjoint

Canonical identifiers

cmha--AHMU1218001
aneux--p043_HAAR…
benchanxplore--MESH_1

동일 환자 여부가 문서화되지 않은 서로 다른 dataset ID는 결합하지 않는다.

Field provenance

06 / EVIDENCE & LIMITATIONS

Promising method. Early clinical evidence.

What is supported

GNN은 불규칙한 CFD mesh를 직접 다룰 수 있고, inlet context와 physics-constrained loss는 장기 rollout 안정성에 기여할 수 있다.

What is not yet supported

semi-idealized geometry의 benchmark 성능을 실제 다기관 임상 위험 예측 성능으로 일반화할 수 없다.

What this project will test

실제 CFD와 surrogate CFD를 분리 비교하고, split·calibration·uncertainty·failure case를 함께 보고한다.

Primary reference: Lannelongue et al., npj Digital Medicine (2026)

07 / LIVING ROADMAP

Evidence accumulates one audited step at a time.

  1. NOW공개 raw archive를 서버에 수집하고 checksum·실제 file count·배포 구조를 inventory한다.
  2. NEXT각 dataset adapter로 clinical column, mesh unit, HDF5 field schema를 명시한다.
  3. REPRODUCE동일 split과 boundary mask에서 MGN → In-PI-MGN ablation을 재현한다.
  4. EXTENDreal CFD와 patient-specific geometry로 domain gap·surrogate uncertainty를 측정한다.
  5. TRANSLATE기관 분리 multimodal risk study에서 calibration과 decision impact를 검증한다.
