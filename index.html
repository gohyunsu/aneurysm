01 / RESEARCH QUESTION

Can we bring CFD-level insight into a clinically usable time scale?

Intracranial aneurysm research has two connected bottlenecks: reliable lesion interpretation from CTA/MRA, and the cost of computing patient-specific hemodynamics. This project studies both, while keeping their labels, data sources, and claims separate.

Working hypothesisClinical + morphology + hemodynamics may provide a richer representation of aneurysm status than static morphology alone—but this must be tested with patient-level external validation.
CTA / MRA→Geometry→Mesh graph→In-PI-MGN→WSS / OSI

02 / TASK DEFINITION

Separate the tasks before combining them.

A

Clinical imaging AI

Detect, segment, and assign vessel-specific location labels to aneurysms in CTA/MRA volumes.

Input
3D CTA or MRA
Output
Lesion mask + vessel/location class
Primary metrics
Lesion sensitivity, Dice, NSD, FP/case, location accuracy
B

Hemodynamics surrogate

Predict time-resolved velocity fields on an irregular CFD mesh without rerunning the full solver.

Input
Mesh graph, state, inlet context
Output
Velocity rollout → WSS / OSI / flow summaries
Primary metrics
1-step/50-step RMSE, WSS error, mass-flux error
C

Risk stratification

Test whether hemodynamic representations improve association with rupture status or clinically relevant outcomes.

Input
Clinical + morphology + real/surrogate hemodynamics
Output
Calibrated risk score, not a treatment decision
Primary metrics
AUROC/AUPRC, calibration, decision-curve analysis
Scope guardrail Ruptured versus unruptured status is not the same as prospective future rupture risk. The site will label these claims explicitly.

05 / EXPERIMENT MATRIX

Make every improvement earn its place.

SURROGATE ABLATION

MGN → PI-MGN → In-MGN → In-PI-MGN

Keep split, normalization, boundary masks, and training budget fixed. Report 1-step and 50-step behavior separately.

RISK ABLATION

Clinical → morphology → + real CFD → + surrogate CFD

Measure incremental value, calibration, missing-modality behavior, and whether surrogate uncertainty changes decisions.

GENERALIZATION

In-domain → geometry/site/modality OOD

Do not call a four-case zero-shot test external validation. Report anatomy, scanner, inflow, and site shifts explicitly.

06 / EVIDENCE & LIMITATIONS

Promising method. Early evidence.

Why it is promising

GNNs operate directly on irregular CFD meshes. Inlet context addresses long-range temporal dependence, while discrete physics terms encourage physically plausible rollouts.

What the paper shows

In-PI-MGN improves long rollout error over the reported MGN baseline and predicts WSS patterns efficiently on the benchmark. The model can run 80 frames in under a minute in the reported setting.

What remains open

The benchmark uses semi-idealized ICA geometries and idealized inflow. Patient-specific boundary conditions, broader anatomy, solver uncertainty, and prospective clinical outcomes remain open.

Primary reference: Lannelongue et al., npj Digital Medicine (2026)
